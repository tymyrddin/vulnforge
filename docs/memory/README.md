# Memory

This directory contains the project's institutional memory: design decisions,
threat models, and architectural debates that explain *why* vulnforge is built
the way it is.

These documents are **preserved**, not updated. When a design decision changes,
we add a new document rather than rewriting the old one. This ensures the
project's reasoning trail remains intact and verifiable.

If you're looking for what we're working on *now*, see [../roadmap.md](roadmap.md).
If you're looking for how to *use* the system, see [../README-technical.md](README-technical.md).
